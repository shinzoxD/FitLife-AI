# FitLife Resume Notes

## One-Line Summary

FitLife is a full-stack fitness and nutrition app that combines OCR-based food label scanning, pose-based workout analysis, and an AI meal coach into one product workflow.

## Resume Bullets

- Built `FitLife`, a full-stack wellness application using Next.js 15, React 19, Flask, and JWT auth, with integrated nutrition scanning, workout analysis, and AI coaching workflows.
- Implemented an OCR-driven nutrition pipeline and a retrieval-backed meal coach that turned user profile data and food inputs into personalized scoring and guidance.
- Shipped a production-style architecture with a Next.js frontend, Flask API gateway, async-ready workout processing, deployment configs for Vercel and Render, and portfolio-focused UX polish.

## Portfolio Description

FitLife is a polished full-stack product concept that demonstrates frontend design work, API integration, authentication, AI-assisted product features, and deployment-ready project structure in a single repo.

## Interview Talking Points

- Why the app is split between a Next.js frontend and a Flask gateway.
- How OCR nutrition extraction and scoring are modeled as a simple user flow.
- How workout analysis is designed to support async processing and polling.
- Why JWT auth was chosen for the frontend app experience.
- How the project balances product design, AI integration, and full-stack delivery in one codebase.
